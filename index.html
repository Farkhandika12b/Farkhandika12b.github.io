HTML

HTML (Hypertext Markup Language) adalah bahasa yang digunakan untuk membuat dan mengatur konten di web. Ini adalah fondasi dari hampir semua halaman web yang Anda temui. Artikel ini akan membahas dasar-dasar HTML, strukturnya, dan bagaimana Anda dapat menggunakannya untuk membuat halaman web sederhana.

Apa itu HTML?

HTML adalah bahasa markup, yang berarti itu digunakan untuk menandai (markup) konten dalam dokumen web. Markup adalah cara kita memberi tahu browser web tentang jenis konten apa yang ada dalam dokumen dan bagaimana menampilkannya. Misalnya, dengan HTML, kita dapat memberi tahu browser bahwa suatu teks adalah judul, paragraf, gambar, atau tautan.
